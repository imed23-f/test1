Présentation de quelques outils pour éditer du code HTML

Le développement des sites Web nécessite des petites interventions sur le code. En particulier pour éditer un code HTML, on cite quelques outils.

Cinq (05) outils :
Notepad++
UltraEdit
EditPad Lite
EditPlus
Visual Studio Code
  • Tableau récapitulatif

    • Notepad++:

    Notepad ++ est un éditeur de code source gratuit. Fonctionnant dans l'environnement Windows, son utilisation est régie par la licence publique générale GNU. Notepad ++ est écrit en C ++ et il utilise l'API Win32 pure et la STL qui assure une vitesse d'exécution plus élevée et une taille de programme plus petite.

    Il a été développé par Don Ho, un informaticien diplômé de l'université de Paris VII - Diderot en 2000.

    Principales fonctionnalités

    1. Coloration syntaxique personnalisable et imprimable
    2. Plier/déplier de portions de code
    3. Complétion automatique
    4. Plusieurs fonctionnalités peuvent être ajoutées grâce aux macros et plugins

    Forces

    1. La conversion binaire et l'autocomplétion
    2. Fonctions de recherche et de remplacement évoluées
    3. Ouverture de plusieurs documents simultanément avec un système d'onglets
    4. Existe en version classique avec installation ou en version portable
    5. Installation des plugins possible
    6. Disponible gratuit en terme de licence
    7. Une simplicité suffisante pour les débutants sur des langages de programmation

    Limites

    1. Mono plateforme, seulement version sous windows disponible

    UltraEdit:

    UltraEdit est un éditeur de texte créé par IDM Computer Solutions. Disponnible en plusieurs langues dont le français.

    Principales fonctionnalités

    1. La coloration syntaxique de code source
    2. Les expressions rationnelles permettant la recherche et le remplacement du texte
    3. La comparaison de fichiers pour détecter les différences
    4. La vérification orthographique des textes et certains reformatages (gestion des éspaces et tabulations)
    5. La définition manuelle ou l'enregistrement de macros
    6. Correcteur orthographique (plusieurs versions nationales disponibles)

    Forces

    1. Multi-platefore, disponible sous Windows, Mac et Linux
    2. L'édition de fichiers distants (par FTP ou par SSH)
    3. L'affichage et l'édition de fichier binaire en hexadécimal ou en format textel
    4. Le reformatage de paragraphes
    5. Les conversions entre plateformes et codages (DOS, MAC, UNIX, ASCII, EBCDIC, OEM, ANSI)
    6. Moteur de scripts

    Limites

    1. Supporte l'édition de fichiers de taille virtuellement illimitée

    EditPad Lite:

    EditPad Lite est la version allégée et gratuite (pour une utilisation personnelle) du logiciel payant EditPad Pro. Version allégée, mais bien suffisante pour un usage d'édition basique. L'interface du logiciel est en français.

    Il est aussi un éditeur de texte compact à usage général. On l'utilise pour modifier facilement tout type de fichier texte brut. EditPad Lite possède toutes les fonctionnalités essentielles pour simplifier l'édition de texte.

    Principales fonctionnalités

    1. Support de fichiers volumineux et de longues lignes
    2. Prise en charge complète d'Unicode, y compris des scripts complexes et des scripts de droite à gauche
    3. Édition directe de fichiers texte à l'aide des encodages de texte (pages de codes) et des sauts de ligne Windows, UNIX et mac Mac
    4. Interface à onglets pour travailler avec de nombreux fichiers
    5. Annulation et rétablissement illimités pour tous les fichiers ouverts, même après l'enregistrement
    6. La sauvegarde automatique et les copies de travail empêchent la perte de données

    Forces

    1. Disponible en plusieurs langues
    2. Application allégée et gratuite

    Limites

    1. Mono plateforme, seulement version sous windows disponible
    2. Editeur de texte basique

    EditPlus:

    EditPlus est un logiciel d'édition de texte moins perfectionné que Notepad++, et son interface est en anglais. Sa simplicité est néanmoins appréciable, ainsi que sa rapidité, et il recèle aussi quelques petites fonctions intéressantes et originales.

    Principales fonctionnalités

    1. On peut sélectionner rapidement, avec un raccourci, tout le texte entre deux balises
    2. On peut fusionner des lignes, par exemple de commentaires
    3. Permet de faire lire un texte sélectionné par l'ordinateur (fonction Text to Speech)
    4. La coloration syntaxique pour le HTML, JavaScript, CSS, PHP, Java, C/C++ et VBScript

    Forces

    1. On peut avec un raccourci clavier, localiser facilement le tag HTML fermant (ou ouvrant) dans une portion de code
    2. La coloration syntaxique fonctionne pour le HTML et JavaScript
    3. Lorem ipsum dolor sit amet

    Limites

    1. Mono plateforme, seulement version sous windows disponible
    2. Editeur de texte basique

    Visual Studio Code:

    Visual Studio Code est un éditeur de code extensible développé par Microsoft

    Il est présenté lors de la conférence des développeurs Build d'avril 2015.

    Il est basé sur Electron. Bien qu'il utilise le framework Electron, le logiciel n'utilise pas Atom mais utilise le même composant éditeur (nommé "Monaco") utilisé dans Azure DevOps (anciennement appelé Visual Studio Online et Visual Studio Team Services).

    Principales fonctionnalités

    1. Il est utilisé pour développer des applications Web Node.js qui s'exécutent sur le moteur de présentation Blink
    2. Il prend immédiatement en charge presque tous les principaux langages de programmation. Plusieurs d'entre eux sont inclus par défaut
    3. Il comprend plusieurs extensions pour FTP, permettant au logiciel d'être utilisé comme une alternative gratuite pour le développement Web
    4. Visual Studio Code collecte les données d'utilisation et les envoie à Microsoft

    Forces

      Véritable couteau suisse, non seulement pour les programmeurs, mais pour tout le monde :
    1. Un éditeur de code multi-plateforme Windows, Mac et Linux
    2. Open source et gratuit
    3. Il supporte une dizaine de langages Java, JavaScript, Go, Node.js, C ++ ...
    4. Le code peut être synchronisé entre l'éditeur et le serveur, sans télécharger de logiciel supplémentaire
    5. Outil de débogage digne de la tradition Visual
    6. Intégration Git parfaite grâce notamment à l’extension “GitLens”

    Limites

      C’est un outil qui possède très peu de limites, il peut être un outil de rédaction technique, un bloc-notes, un éditeur de brouillons, un terminal CMD, PowerShell, SSH /Linux entre autres.
